Password Strength Meter - jQuery plugin

Password Strength Meter is a jQuery plugin that allows you to easily bind a progress bar showing the strength of a password to your inputs with multiple options.

Check the docs for more info.

Just the password input, with the progress shown since the very first moment
Default behavior (progress bar animated on focus/blur)
Linked to username input (password cannot contain field text)
Change translations
Custom events (button enabled when score > 75%)
Using input-groups
